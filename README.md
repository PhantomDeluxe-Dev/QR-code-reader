# 📱 QR Code Reader

A simple, browser-based QR code reader built with HTML & JavaScript. No installation required — just open the file and start scanning!

🔗 **Live Demo:** [Try it here](https://phantomdeluxe-dev.github.io/QR-code-reader/)

## ✨ Features

- 📷 Scan QR codes directly from your browser
- ⚡ Lightweight — no build tools or dependencies to install
- 🌐 Runs entirely on the client side (HTML/JS)

## 🚀 Getting Started

No installation needed! Just:

1. Clone this repository
```bash
   git clone https://github.com/PhantomDeluxe-Dev/QR-code-reader.git
```
2. Open `index.html` in your browser
3. Allow camera access and start scanning QR codes!

- HTML5
- JavaScript
